# Vigenere Cipher

## Description
The Vigenère cipher is a polyalphabetic encryption method that uses a keyword to encrypt the text

## Example:
>**Message:** HELLO
>**Key:** KEY (repeated up to the length of the message: KEYKE)
>>**Encryption:**
>> - H + K (shift by 10) → R
>> - E + E (shift by 4) → I
>> - L + Y (shift by 24) → J
>> - L + K (shift by 10) → V
>> - O + E (shift 4) → S
>>Encrypted message: RIJVS
